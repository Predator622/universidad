### 1. Lista de Archivos y Roles

**Archivos:**
- `main.cpp`: Contiene la función principal que maneja la entrada del usuario, lectura del archivo, y llamadas a funciones para buscar palabras.
- `utils.h`: Encabezado con declaraciones de utilidades como funciones auxiliares y estructuras de datos necesarias.
- `utils.cpp`: Implementación de las funciones definidas en `utils.h`.
- `buscador_palabras.h`: Definición del TAD (Tipo Abstracto de Datos) para el buscador de palabras, incluyendo métodos para buscar por subcadena al inicio, dentro y invertida.
- `buscador_palabras.cpp`: Implementación del TAD definido en `buscador_palabras.h`.

**Roles:**
- **main.cpp**: Maneja la entrada/salida estándar, lectura de archivos y ejecución de las funciones principales.
- **utils.h/cpp**: Contiene funciones auxiliares para procesamiento de cadenas y estructuras de datos necesarias.
- **buscador_palabras.h/cpp**: Implementa el TAD que realiza la búsqueda de palabras en función de subcadenas.

### 2. Decisiones Técnicas Clave

1. **Uso de Estructuras Lineales:** Se utilizaron `std::list` para almacenar las líneas del archivo y `std::stack` para manipular cadenas durante el procesamiento.
2. **Lectura de Archivos:** La lectura se realiza línea por línea, asegurando que cada línea sea tratada individualmente antes de buscar subcadenas.
3. **Validaciones:** Se implementaron validaciones básicas en la función principal para manejar errores comunes como archivos no existentes o cadenas vacías.

#### `main.cpp`
#include <iostream>
#include <fstream>
#include "utils.h"
#include "buscador_palabras.h"

int main(int argc, char* argv[]) {
    if (argc != 3) {
        std::cerr << "Uso: ./mi_programa archivo.txt subcadena" << std::endl;
        return -1;
    }

    std::string filename = argv[1];
    std::string substring = argv[2];

    // Leer el archivo y procesar las líneas
    std::vector<std::pair<int, std::string>> lines = read_file(filename);

    BuscadorPalabras buscador(substring);
    
    // Búsqueda de palabras que comienzan con la subcadena
    auto resultados_inicio = buscador.buscarPorInicio(lines);
    imprimirResultados(resultados_inicio, "Comienzan por");

    // Búsqueda de palabras que contienen la subcadena
    auto resultados_contienen = buscador.buscarContiene(substring, lines);
    imprimirResultados(resultados_contienen, "Contienen");

    // Búsqueda de palabras que contienen la subcadena invertida
    std::string substring_invertido = invertirCadena(substring);
    auto resultados_invertidos = buscador.buscarInverso(substring_invertido, lines);
    imprimirResultados(resultados_invertidos, "Contienen Invertido");

    return 0;
}
