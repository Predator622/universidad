package com.quantum_simulation.registry;

import java.util.Map;
import java.util.concurrent.ConcurrentHashMap;

public class ComponentRegistry {
    private static final Map<Class<?>, Object> registryMap;

    static {
        registryMap = new ConcurrentHashMap<>();
    }

    public static <T> void register(Class<T> clazz, T instance) {
        registryMap.put(clazz, instance);
    }

    @SuppressWarnings("unchecked")
    public static <T> T getRegisteredInstance(Class<T> clazz) {
        return (T) registryMap.get(clazz);
    }
}

# Quantum Simulation Project

## Overview
This project is a simulation engine for quantum and fluid dynamics. It uses the Singleton pattern to manage global state and the Registry pattern to register components.

## Setup
1. Clone the repository.
2. Run `mvn clean install` to build the project.
3. Execute the main class from the `core` package.

## Configuration
Configuration files are located in `src/main/resources/configuration`.

## Contributing
Feel free to contribute by submitting pull requests or issues.
