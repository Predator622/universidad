### 1. Lista de Archivos

- `taller_sql.pdf`: Documento PDF con respuestas a las preguntas, código SQL y resultados en pantalla.
- `script.sql`: Script SQL que contiene todas las consultas realizadas para el taller.
- `01-DDL.sql`: DDL (Data Definition Language) para crear estructura de base de datos.
- `02-Data_Insert.sql`: Data Insert para poblar la base de datos con datos iniciales.

### 2. Decisiones Técnicas Clave

- **Uso de Oracle SQL Developer**: Se seleccionó esta herramienta por su facilidad en manejar scripts y visualizar resultados directamente.
- **Estructura Modular del Script .sql**: Cada sección (descripción, verificación, consultas) está bien organizada para facilitar la revisión y mantenimiento.
- **Incluir Pantallazos de Resultados**: Para cada consulta compleja o relevante, se incluirán pantallazos que muestran los resultados obtenidos.
